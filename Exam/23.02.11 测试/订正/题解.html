T1

题意

给你一棵树上每个节点的度，输出这棵树的一种可能的直径或者当无解时输出 −1-1−1。

数据范围

n≤20n \le 20n≤20

思路

本题为结论题，并且正解时间复杂度严格线性，不知道为什么这道题数据范围只给到了 n≤20n \le 20n≤20。

Lemma 1：任何一棵节点数为 nnn 树中的节点度数之和为 2×(n−1)2 \times (n - 1)2×(n−1)。

我们知道，节点数为 nnn 的树中共有 n−1n - 1n−1 条边，每条边对度数的贡献是 222，于是引理显然成立。

这个引理的作用是帮助我们快速判断题目中给出的数据是否有解。

Lemma 2：存在一种建树方法，使得所有非叶子节点都位于树的直径上。

可以通过数学归纳法证明。

比如说，我们有如下的一棵树：

设点 222 的度数为 d2d_2d2​，点 333 的度数为 d3d_3d3​，我们可以通过如下操作，使得它们的度数不变，而将 222 与 333 都转移到同一条链上。为简化问题，我们将 333 转移到 222 的下方。

  1. 断开 222 与其子节点中的一个，记为 son2son_2son2​，的连接；
  2. 取 333 本身以及 d3−2d_3 - 2d3​−2 个子节点，使其成为 222 的新的子节点，此时 222 的度数还是 d2d_2d2​；
  3. 将 333 剩余的一个子节点，记为 son3son_3son3​，挂到 333 原本的位置上，再使 son2son_2son2​ 成为 333 的子节点，此时，333 的度数为 d3−1+1=d3d_3 - 1 + 1 = d_3d3​−1+1=d3​。

最终的效果如下：

我们可以通过相似的操作，使得每个非叶子节点都汇到一条链上，那么上述引理成立。

如果我们通过上述方法建树，这棵树的直径就是 d=∣{t∣t为非叶子节点}∣+1d = |\{t | t 为非叶子节点\}| + 1d=∣{t∣t为非叶子节点}∣+1。

注意在出现负度数的时候输出 −1-1−1。

#include <iostream>

	ios::sync_with_stdio(false);
	cin.tie(0), cout.tie(0);
	
    cin >> n;
    for (int i = 1; i <= n; i++) cin >> d[i], tot += d[i];

    if (tot != (n - 1) << 1) cout << -1 << '\n';
    else {
        for (int i = 1; i <= n; i++) 
            if (d[i] >= 2) ans++;
            else if (d[i] <= 0) return cout << -1 << '\n', 0;
        cout << ans << '\n';
    }

	return 0;
}

T2

题意

给你一个矩阵 AAA 如下：

[abcd]\begin{bmatrix} a & b \\ c & d \\ \end{bmatrix}[ac​bd​]

现在你要构造一个矩阵 BBB：

[a1b1c1d1]\begin{bmatrix} a_1 & b_1 \\ c_1 & d_1 \\ \end{bmatrix}[a1​c1​​b1​d1​​]

满足 a1×d1=c1×d1a_1 \times d_1 = c_1 \times d_1a1​×d1​=c1​×d1​ 且 A−BA - BA−B 中的绝对值最大元素最小。

思路

最大值最小，不难想到二分答案。

我们考虑二分一个答案 ansansans，那么有 a1∈[a−ans,a+ans]a_1 \in [a - ans, a + ans]a1​∈[a−ans,a+ans]，其余三项同理。如果答案小于等于 ansansans，那么满足 {m∣m=a1×d1}∪{n∣n=b1×c1}≠∅\{m | m = a_1 \times d_1\} \cup \{n | n = b_1 \times c_1\} \neq \emptyset{m∣m=a1​×d1​}∪{n∣n=b1​×c1​}=∅ 即可。

代码

注意处理含负数时的集合边界即可。

#include <bits/stdc++.h>

static const double EPS = 1e-6; 

using namespace std;
double a, b, c, d;
double ans;

bool check(double mid) {
	double l1 = min(min((a - mid) * (d - mid), (a + mid) * (d + mid)), 
					min((a - mid) * (d + mid), (a + mid) * (d - mid)));
	double r1 = max(max((a - mid) * (d - mid), (a + mid) * (d + mid)), 
					max((a - mid) * (d + mid), (a + mid) * (d - mid)));
	
	double l2 = min(min((b - mid) * (c - mid), (b + mid) * (c + mid)), 
					min((b - mid) * (c + mid), (b + mid) * (c - mid)));
	double r2 = max(max((b - mid) * (c - mid), (b + mid) * (c + mid)), 
					max((b - mid) * (c + mid), (b + mid) * (c - mid)));
	
	if (r1 + EPS <= l2 || r2 + EPS <= l1) return false;
	else return true;
}

int main() {
	ios::sync_with_stdio(false);
	cin.tie(0), cout.tie(0);

	cin >> a >> b >> c >> d;

	double l = 0, r = 2e9;
	while (l + EPS <= r) {
		double mid = (l + r) / 2;
		if (check(mid)) ans = mid, r = mid;
		else l = mid;
	}

	cout << fixed << setprecision(3) << ans << '\n';
	return 0;
}
